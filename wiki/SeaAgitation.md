# 海风程度  
变化范围: 0 ~ 5  
基础变化率: 0 / 每15分钟  
## 阶段  
范围  |  名称  |  描述  |  影响状态  |  影响动作  
----  |  ----  |  ----  |  ----  |  ----  
0 ～ 0  |  平静的水面  |    |  [晕船](SeaSickness.md)加成-10  |    
1 ～ 1  |  轻微的浪  |    |  [晕船](SeaSickness.md)加成-1  |    
2 ～ 2  |  中等的浪  |    |  [晕船](SeaSickness.md)加成+2  |    
3 ～ 3  |  强力的浪  |    |  [晕船](SeaSickness.md)加成+3  |    
4 ～ 5  |  风暴的海域  |    |  [晕船](SeaSickness.md)加成+4  |
