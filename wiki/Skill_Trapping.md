# 陷阱(技能)  
> 你的陷阱技能决定了你可以建造哪些陷阱蓝图，也影响你真正捕获到猎物的<b>概率</b>。<br><br>大多数陷阱需要<b>诱饵</b>才能工作。使用合适的诱饵来引诱你想要的<b>动物</b>。<br><br>除非真的有必要，否则尽量不要更换<b>诱饵</b>，这样做会吓跑动物并重置陷阱的计时器。  
  
变化范围: 0 ~ 150  
基础变化率: 0 / 每15分钟
